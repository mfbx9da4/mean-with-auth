MEAN Boilerplate with Email Auth
==============================

## Install
    
- Requires node v6. Use [nvm](https://github.com/creationix/nvm) for example, for managing versions.
- Requires mongodb installed and running (`mongod`).

    npm install

## Test and coverage

Test server

    npm run test:server

Test client

    npm run test:client

## Run app
    
    npm run start
